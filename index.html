WHAT I BUILD
Minecraft · C++ · Windows
MinecraftConsoles

A fork of smartcmd/MinecraftConsoles focused on stable, tested code. Based on Minecraft Legacy Console Edition v1.6.0560.0 (TU19). No nightly builds — weekly releases with improvements or fixes only.

KEY FEATURES
  • + Fixed compilation Debug & Release (VS 2022)
  • + Keyboard & mouse input support
  • + Fullscreen toggle (F11) + native resolution
  • + LAN multiplayer & discovery (TCP 25565 / UDP 25566)
  • + Persistent username via username.txt
  • + High-res timer for smooth high-FPS timing

⚠ SECURITY — LAN RCE
Known RCE vulnerability on Windows via LAN. Use ZeroTier or Radmin VPN. Play on open LAN at your own risk.
